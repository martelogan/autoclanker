# Portfolio Mapping

Use these rules when converting idea families into a big-bet registry.

- A big bet is a strategic theme with a near-term win and a longer-term unlock.
- An idea family is an issue-backed exploration lane.
- An idea family belongs to exactly one big bet.
- Big bets should be few enough to discuss in a meeting without scrolling.
- Dependencies are explicit edges, not implied by drag order.
- `P0` is the current front; later `P` layers should still have concrete next
  actions when they are near enough to discuss.
- `locked` means earlier evidence or contracts are still missing.
- `emerging` means useful exploration can begin, but the unlock is not proven.
- `unlocked` means prior work produced durable evidence that changes what can
  be attempted next.
- If a lane has no current evidence bundle, seed artifact, or kickoff path, keep
  it as `candidate`/`follow-up` until the authoring gap is closed.
- If a lane has partial success, split the shipped subset from the remaining
  exploration only when the remaining proof path is materially different.
